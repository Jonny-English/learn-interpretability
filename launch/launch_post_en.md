# Learn Interpretability Is Live

I’ve open-sourced **Learn Interpretability**, a bilingual (`English` / `中文`), Colab-first interpretability course built for serious self-study.

This repo is aimed at readers who already know basic Python/PyTorch but are new to mechanistic interpretability and want more than a paper list. The structure is:

- foundation labs for setup, plots, geometry, and experiment discipline
- one-note-per-paper coverage of the Anthropic interpretability arc
- honest live notebooks only where the repo can generate outputs at runtime
- extension reproductions across transformer circuits, memory/editing, and auditing
- a research-ready layer with memos, critiques, rubrics, dossiers, and graduation gates

The main constraint I enforced is honesty:

- runnable means live generation at runtime
- anything that would depend on unavailable stacks or precomputed artifacts is labeled `reading-only`
- the repo does not claim “finish 12 weeks and become senior”

Instead, the direct promise is narrower and more useful: if a learner finishes the full path and clears the evidence gates, the repo is designed to move them toward a credible `P6-ready` / early-`P6` state.
