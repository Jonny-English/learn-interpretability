# Learn Interpretability 正式开源

我把 **Learn Interpretability** 开源了。这是一个双语（`English` / `中文`）、Colab 优先的 interpretability 教学仓库，面向已经有基础 Python / PyTorch、但刚进入 mechanistic interpretability 的学习者。

它不是论文清单，而是一条完整训练路径：

- foundations：补环境、图表、几何和实验纪律
- Anthropic interpretability 主线：一篇文章一份讲义
- 只有在运行时能实时产生输出的内容，才给 live notebook
- 扩展复现：补 transformer circuits、memory/editing、auditing
- research-ready 层：memo、critique、rubric、dossier、graduation gates

这套仓库最重要的一条规则是“诚实”：

- runnable 就是真的实时可跑
- 需要预计算 artifact 或不可公开栈的内容，直接标成 `reading-only`
- 不再包装成“学完 12 周就自动成为高阶研究者”

更实际的目标是：
如果学习者完成完整路径，并通过证据门槛，这套仓库会把他推进到一个可信的 `P6-ready / 早期 P6` 状态。
