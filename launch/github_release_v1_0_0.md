# Learn Interpretability v1.0.0

`Learn Interpretability` is now publicly launch-ready as a bilingual, Colab-first interpretability course for serious self-study.

## What this project is

- A bilingual (`English` / `中文`) interpretability training repo
- Colab-first, with runnable notebooks only where the repo can be honest about live generation
- A structured path from foundations to a `P6-ready` research workflow

## What is in v1

- `F00-F03` foundation labs for environment setup, plots, vector geometry, and experiment discipline
- `M00-M10` article notes, with live notebooks for the runnable core and reading-only handling for non-live items
- `X01-X07` extension reproductions across transformer circuits, memory/editing, and auditing
- A research-ready layer with memos, rubrics, sprints, dossiers, and graduation gates
- Mirrored English and Chinese docs and notebooks

## What this project does not claim

- It does not claim that reading the notes or finishing 12 weeks alone makes you research-ready
- It does not claim full live reproduction for papers that would require unavailable stacks or precomputed artifacts
- It does not claim to replace advisors, collaborators, or long-horizon apprenticeship

## The honest promise

If a learner with the stated prerequisites completes the full path, including extension reproductions and evidence gates, the repo aims to get them to a credible `P6-ready` / early-`P6` state as defined in the documentation.

## Start here

- New to notebook workflows or experiment logging: start with `F00 -> F03`
- Already comfortable with PyTorch and Colab: start with `M00 -> M05`
- Want the full research path: read the `Research-ready` overview and the `P6 graduation checklist`

## Highlights

- Bilingual and structurally mirrored
- Colab-first and CPU-friendly where possible
- Strict honesty rule for live vs reading-only content
- Validation, smoke tests, link checks, and realtime audit scripts included
