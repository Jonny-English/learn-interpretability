# M00 视觉电路热身

## 目标

在进入语言模型之前，先把三个直觉建立起来：单个神经元可能对应可识别的模式，多个神经元会组成电路，而一旦电路变得可见，干预也会更容易推理。

## 核心参考

- [Zoom In: An Introduction to Circuits](https://distill.pub/2020/circuits/zoom-in/)

## 重点观察

- 为什么在视觉模型里，单个神经元的可视化就已经足够直观。
- 为什么多个边缘探测器可以组合成一个曲线探测器。
- 为什么这套视觉故事非常有帮助，但不足以直接解释语言模型。
