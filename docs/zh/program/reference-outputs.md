# 参考答案层

这不是让你抄模板，而是让你校准“什么叫一个能交给导师、协作者或同行评审看的输出”。

## 这层解决什么问题

- 很多初学者知道自己要写 memo，但不知道“写到什么密度才算够”。
- 只有 rubric 没有样例时，最容易出现“我写了很多字，但还是没有判断”的问题。
- 参考样例的作用不是替你思考，而是让你看到证据、风险、判断和 next step 应该怎样排布。

## 推荐使用方式

- 先自己写一版，再去对照样例，不要先看答案。
- 看样例时重点看结构密度，而不是抄措辞。
- 每次对照都问一句：我的输出缺的是证据、判断、风险，还是下一步计划？

## 什么时候最该用

- 第一次做独立研究冲刺之前。
- 第一次交 memo 之前。
- 你觉得自己“写了很多”，但别人还是看不出判断时。
