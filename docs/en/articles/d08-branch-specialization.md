## Summary

Measure how different Inception branches respond to line, arc, and frequency families, then render branch-specific exemplars live.

## Notebook Scope

The notebook studies specialization as a live branch-by-family response pattern rather than as a static diagram imported from the paper.

## Live Coverage

- Branch-wise response heatmap across stimulus families
- Top-channel search inside each branch
- Live feature renders for branch-specific exemplars

## Integrity Note

The specialization story here comes from live measurements on public branches and stimuli. It is meant to be transparent and inspectable rather than exhaustively paper-faithful.
