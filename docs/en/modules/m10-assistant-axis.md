# M10 The Assistant Axis

## Goal

Study how assistant styles can be placed on a shared axis and why stabilizing that axis matters.

## Core reference

- [The Assistant Axis](https://www.anthropic.com/research/assistant-axis)

## What to look for

- Multiple assistant styles can be compared in one shared frame.
- Helpfulness, safety, and warmth do not move independently.
- Stabilizing character is an interpretability problem and a product problem at the same time.

## Status

- `reading-only` under the strict realtime policy

## Self-check questions

- Why is it useful to compress many assistant styles onto one axis, yet still insufficient?
- Which properties seem to change smoothly along the axis, and which ones clearly do not?
- If you had two weeks to study character stabilization, what smallest useful experiment would you design around this axis?

## Takeaway

The paper turns character control from a local intervention question into a broader stability question.
