# M05 Character and Control

## Goal

Connect interpretability to character and behavior by studying persona vectors and two frontier reading prompts.

## Core references

- [Persona vectors](https://www.anthropic.com/research/persona-vectors)
- [Signs of introspection in large language models](https://www.anthropic.com/research/introspection)
- [The assistant axis](https://www.anthropic.com/research/assistant-axis)

## What to look for

- Character traits can be monitored as directions rather than only as prompt-level effects.
- Steering a persona is not the same thing as solving alignment or self-knowledge.
- Frontier papers should widen questions, not create false confidence.

## Notebook and artifacts

- Notebook: `notebooks/en/m05_character_control.ipynb`
- Shared artifact: `artifacts/m05_persona_vectors.json`

## Takeaway

Interpretability can reach into model character, but the resulting control problems are still frontier research questions.
