# P8 Roadmap

## First, what this is not promising

- This repository does not promise "finish 12 weeks and automatically become P8."
- The more accurate statement is: it is designed as a training system that points toward the long-horizon goal of an Alibaba-style `P8` interpretability researcher.
- The 12-week core bootcamp is for entry usefulness; P7/P8 requires a longer cycle of extension reproductions, writing, proposals, and independent judgment.

## The growth arc from P4 to P8

- `P4`: can run notebooks under guidance, restate concepts, and record basic experiments.
- `P5-P6`: can independently reproduce scoped papers, write baselines and failure analysis clearly, and take on explicit research tasks.
- `P7`: can connect multiple papers into one direction, propose a two-week research plan, and discuss evidence boundaries plus tooling needs.
- `P8`: can own an interpretability sub-area or tooling line and create stable technical leverage for multiple collaborators.

## Which repository layer trains which segment

- `F00-F03` mainly serve `Pre-P4 -> P4`.
- `M00-M10` mainly serve `P4 -> P5/P6`.
- `X01-X07`, the reference-output layer, the company onramp, and repeated capstones mainly serve `P6 -> P7/P8`.

## If your real goal is P8

- Do not stop at the core; the core is the floor.
- You need to turn the extension papers into Colab reproductions rather than only reading summaries.
- You need to accumulate a portfolio, not just completions.
- You need repeated practice writing the next experiment, because the heart of P8 is not only plot reading; it is problem selection and direction-setting.

## What to watch if you want to know whether you are still growing upward

- Whether you can place multiple papers inside one research language.
- Whether you can state the evidence boundary of an explanation rather than only restating the author's conclusion.
- Whether you can propose a two-week plan that a real team could execute.
- Whether you can put interpretability, editing, auditing, and control inside one judgment frame.
