# First-Week Proof Path

Use this file if you want to answer one question before investing deeply:

Can this repository actually work for me?

This is a one-week proof-of-fit path for a beginner or undergraduate who is serious but not yet sure whether they should commit to the full `P6-ready` program.

## What To Do

1. Run [F00](../../../notebooks/foundations/en/f00_environment_plots_baselines.ipynb) in Colab.
2. Run [M00](../../../notebooks/en/m00_zoom_in_circuits.ipynb).
3. Run [M01](../../../notebooks/en/m01_toy_models_superposition.ipynb).
4. Write one short reading note, one experiment log, and one 100-300 word memo.

## What Counts As Success

- You can get through the Colab environment without feeling lost the whole time.
- You can explain, at a high level, what a feature, a circuit, and superposition are.
- You can tell the difference between a baseline, a variant, and a conclusion.
- You can write one short memo that distinguishes observation from inference.
- You still want to continue after doing real work, not just after browsing the README.

## What Counts As A Warning Sign

- You still cannot run the notebooks reliably, even with Colab.
- You can copy outputs but cannot explain what changed and why it matters.
- You cannot write a 100-word summary of what M01 taught you.
- You consistently avoid notes, logs, and memos and only want to look at plots.

## What To Do Next

- If you are still shaky on setup, plots, or experiment discipline, continue with `F01-F03` before pushing harder.
- If `F00`, `M00`, and `M01` felt manageable, continue into the live core: `M02`, `M04`, and `M05`.
- If the repo already feels useful and credible at this point, star it and keep going.

## Honest Decision Rule

Do not ask whether the repository is perfect after one week.

Ask whether it already gives you:

- a real Colab path
- a more coherent learning sequence than random paper-hopping
- a clearer sense of what interpretability work actually looks like

If the answer is yes, the repository is doing its job.
